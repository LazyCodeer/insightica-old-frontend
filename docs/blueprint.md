# **App Name**: Insightica

## Core Features:

- Main Headline Display: Displays the main headline 'Clarity Over Chaos, Insights Over Noise' prominently on the homepage.
- Subheadline Explanation: Showcases a subheadline explaining Insightica's value proposition: clear indicator analysis, optimal pairings, ML forecasting.
- Homepage CTAs: Includes 'Try Now' (early access) and 'Learn More' (tutorial page) call-to-action buttons.
- Core Benefits Display: Visually represents the three core benefits: Intuitive Graphical Views, Machine-Learning Powered, and User Empowerment.
- Key Benefits Visualization: Showcases key benefits with clear descriptions: Wide Indicator Coverage, High Prediction Accuracy, Fast Analysis, Detailed Metrics.
- Evaluator Tool: Evaluator: Generates performance benchmarks of technical indicators using historical market data, compares indicators, and analyzes indicator combinations.
- Predictor Tool: Predictor: Uses a tool to predict the relative importance of technical indicators with confidence scores, and offer recommendations tailored to trading parameters.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082), representing intellect, insight, and sophistication appropriate to a serious tool.
- Background color: Very dark indigo (#1A0033) to ensure legibility.
- Accent color: Purple (#800080) to contrast, for focus and legibility of CTAs.
- Body text: 'PT Sans' (sans-serif) which is also suitable for short headlines. 
- Use minimalist icons to represent data points, indicators, and insights, to fit the aesthetic
- Use a grid-based layout to ensure content is well-organized and easily digestible.
- Incorporate subtle animations on data visualizations, such as line graphs and charts.
